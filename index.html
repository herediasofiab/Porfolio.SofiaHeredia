programmers
Python Project

In this project, I used Python to conduct a comprehensive analysis of 64,000 surveys of programmers from around the world. The main objective was to extract key information related to the requirements to obtain employment as a programmer, identify the most in-demand technologies in the field, and evaluate the employment situation in the industry. In addition, we sought to verify common assumptions and beliefs about programmers and their work environment. The analysis process was divided into six fundamental stages: ask, prepare, process, analyze, visualize and act. To achieve this, data cleaning, statistical measurements and correlation analysis were carried out in order to extract meaningful information and support solid conclusions. The project was developed using the Python programming language, taking advantage of the capabilities of the Numpy, Seaborn and Matplotlib libraries to perform data manipulation, visualization and graph generation tasks.

Legal Document AI Assistants
Project with JSON & OpenAI Agents

In this project, I created AI assistants using OpenAI's platform and JSON to extract structured data from legal documents such as wills, ID cards (DNIs), and deeds uploaded to a web application. Each assistant specialized in understanding and structuring a specific type of document. A final assistant was then built to consolidate all data and automatically generate a legal document called partition notebook, a legal inheritance document detailing the heirs and the value distribution. I designed specialized agents to manage different stages of legal processing using artificial intelligence.

Banking Transaction Automation
with Power Automate & AI

This automation project used Microsoft Power Automate to process incoming bank transaction emails. Each email contained bank receipts in different formats, currencies, and languages from multiple international institutions. I created an AI-based classification system for purchase/sale of bonds and stocks. Extracted data was processed and stored in an Excel sheet, and an automated email response confirmed successful processing. The end-to-end workflow—including intelligent parsing, Excel entry, and notifications—was fully automated.
