Asistentes de IA para Documentos Legales
Proyecto con JSON y Agentes de OpenAI

En este proyecto, creé asistentes de IA utilizando la plataforma de OpenAI y JSON para extraer datos estructurados de documentos legales como testamentos, DNIs y escrituras subidos a una aplicación web. Cada asistente se especializó en comprender y estructurar un tipo de documento específico. Luego se construyó un asistente final para consolidar todos los datos y generar automáticamente un documento legal llamado cuaderno particionario, que detalla herederos y distribución del valor. Diseñé agentes especializados para manejar diferentes etapas del proceso legal mediante inteligencia artificial.

Automatización de Transacciones Bancarias
con Power Automate e IA

Este proyecto de automatización utilizó Microsoft Power Automate para procesar correos electrónicos de transacciones bancarias. Cada correo contenía comprobantes en diferentes formatos, monedas e idiomas de varias instituciones internacionales. Creé un sistema de clasificación basado en IA para operaciones de compra/venta de bonos y acciones. Los datos extraídos fueron almacenados en una hoja de Excel, y se envió una respuesta automática confirmando el procesamiento exitoso. El flujo de trabajo de punta a punta fue completamente automatizado.

Automatización de Registro de Pedidos
con MAKE y Trello

En este proyecto, utilicé la plataforma MAKE para capturar y registrar pedidos de clientes entrantes y enviarlos directamente a tableros de Trello. Esto ayudó a agilizar el flujo de trabajo, reducir entradas manuales y mantener un seguimiento claro de los pedidos. Cada nuevo pedido activaba el flujo automatizado, asegurando actualizaciones rápidas, fiables y estructuradas en tiempo real.

Joyería Inteligente

Tienda de Ropa en Amazon
Dashboard en Power BI

Creé un dashboard interactivo en Power BI para analizar datos de ventas de una tienda de ropa en Amazon. Utilicé herramientas como DAX y Excel para transformar los datos. El tablero muestra visualizaciones detalladas del estado de ventas, unidades vendidas, devueltas y evolución temporal por categoría, con fotos de productos para facilitar la identificación visual.
